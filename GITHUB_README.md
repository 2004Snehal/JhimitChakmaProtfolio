# Snehal Pal - Portfolio Website 🚀

A modern, responsive portfolio website showcasing entrepreneurship journey, AI research, and technical achievements.

## ✨ Live Demo
[View Live Site](https://your-vercel-app-url.vercel.app)

## 🛠️ Built With
- **Frontend**: React 18, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Express.js, Node.js  
- **Database**: Drizzle ORM
- **Deployment**: Vercel

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/SnehalPortfolio.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 📁 Project Structure
```
├── client/          # React frontend
├── server/          # Express backend
├── shared/          # Shared types
├── attached_assets/ # Static assets
└── vercel.json      # Deployment config
```

## 🌐 Deployment
This project is configured for one-click deployment on Vercel:

1. Push to GitHub

3. Deploy automatically!
